## ⌨ Como executar o projeto

```
# Clonar o repositório
$ git clone https://github.com/Mur1lol/Happy-NLW

# Entrar no diretório principal
$ cd Happy-NLW

# Entrar no diretório
$ cd web

# Baixar as dependências
$ yarn install

# Executar o servidor
$ yarn start
```
